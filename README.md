# WBR Intelligence Platform

## Overview

The WBR Intelligence Platform is an automated solution designed to streamline the Weekly Business Review (WBR) process. It eliminates manual data aggregation, enables real-time insights, and empowers leadership with interactive, on-demand analytics using Power BI, MCP integration, and Claude.

## Problem

The traditional WBR process requires manual extraction of data from dashboards, consolidation into Excel sheets, and preparation of reports. This approach is time-consuming, error-prone, and limits the ability to respond to ad hoc questions during leadership reviews. As a result, decision-making can be delayed or based on incomplete information.

## Solution

This platform automates the WBR workflow by integrating a customized Power BI dashboard with an MCP (Model Context Protocol) server and Claude. It enables seamless access to live data and supports natural language queries for real-time insights.

## Key capabilities:

1) Forked and customized Power BI dashboards aligned to WBR needs
2) Integration with MCP server for structured data access
3) Claude-powered natural language querying for dynamic insights
4) Updated memory and optimized prompts for accurate responses
5) Real-time data access during WBR meetings

## Tech Stack

1) Power BI – Data visualization and dashboarding
2) MCP Server – Integration layer for structured data access
3) Claude – AI assistant for natural language querying and insights
4) SQL / Data Warehouse – Data storage and retrieval
5) Cloud Platform (BQ) – Infrastructure and hosting

## Business Impact

1) Eliminates manual data extraction and Excel-based reporting
2) Reduces time and effort required for WBR preparation
3) Enables real-time responses to leadership queries
4) Improves accuracy and consistency of reported metrics
5) Enhances decision-making with instant access to insights
6) Increases productivity of analysts and business teams

## Assumptions

The first version of the solution is already implemented and operational, with users having the necessary access to dashboards and permissions, and data sources assumed to be reliable and up to date. Ongoing improvements will focus on increasing automation and reducing manual dependencies, while continued prompt tuning and memory enhancements will improve accuracy. Adoption is expected to grow steadily as the system proves its reliability and usability.

## Future Enhancements

4) Integration with additional data sources
5) Enhanced UI/UX for broader adoption

## Getting Started

1) Install Claude Desktop App
2) Sign In using analytics.ai@lenskart.com email
3) Complete the sign-in process using the verification code, currently managed and shared by Arun from the Analytics team.
4) Access **Latest WBR creation** chat from the chat section 
5) Ask Claude to connect to the required Power BI dashboards.
6) A browser window will open. Sign in using your Power BI credentials to authorize access.
7) Once connected, ask Claude to generate the latest WBR using the prompts defined in the **prompt** file.
